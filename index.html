Metodologi

01
Pengumpulan Dataset

Sebanyak 1.609 gambar dikumpulkan menggunakan webcam langsung di Google Teachable Machine. Setiap kelas merekam objek dari berbagai sudut, jarak, dan kondisi pencahayaan untuk meningkatkan generalisasi model.

≥400 gambar/kelas
02
Konfigurasi & Training

Model dilatih dengan 100 epoch, learning rate 0.001, dan batch size 16 menggunakan arsitektur MobileNet yang di-fine-tune melalui transfer learning. Seluruh proses berjalan langsung di browser via TensorFlow.js.

MobileNet · Transfer Learning
03
Evaluasi & Pengujian

Setiap kelas diuji real-time via webcam. Seluruh kelas mencapai confidence score 100% tanpa kebingungan antar kelas, menunjukkan tidak ada overfitting maupun underfitting pada model akhir.

Confidence 100% · 4/4 Kelas
04
Export & Integrasi Web

Model diekspor dalam format TensorFlow.js (model.json + weights.bin + metadata.json) dan diintegrasikan langsung ke halaman HTML ini. Inferensi berjalan sepenuhnya di browser tanpa server backend.
